# vue3-jingdong
## 简介

使用vue3开发一个简易的购物应用,主要完成前端部分

**技术栈：vue3**

## 安装

- 获取项目代码

```bash
git clone https://github.com/Blank6682/vue3-jingdong.git
or
git clone git@github.com:Blank6682/vue3-jingdong.git

```

- 安装依赖

```bash
cd vue3-jingdong

yarn install
```

- 运行

```bash
yarn serve
```

- 打包

```bash
yarn build
```
